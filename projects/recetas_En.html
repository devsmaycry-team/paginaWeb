Cooking Recipe Management System

Symfony 5.4 MySQL HTML, CSS and JavaScript

General Description

Online cooking recipe system that allows visitors, whether registered or not, to view recipes. Registered users can upload and edit their own recipes, as well as rate other users' recipes.
