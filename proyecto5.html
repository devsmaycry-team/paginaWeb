Sistema de Gestión de Carnicería

Sistema comercial diseñado para la administración integral de una carnicería. Permite gestionar productos, ventas, clientes, fiado, pedidos y reportes de ganancias desde una única plataforma.

Netbeans Java Java FX MySql

Descripción General

Sistema de gestión orientado a comercios del rubro carnicería, pensado para cubrir todas las tareas diarias del negocio: carga de productos, control de stock, ventas por ticket y administración de clientes.

El sistema permite manejar tanto productos unitarios como carnes y pollos por peso, registrar ventas en tiempo real, calcular ganancias diarias y llevar el control de clientes con cuenta corriente (fiado).

⚙️ Módulos del Sistema

1

Gestión de Productos

Carga y administración de mercadería

Administración completa del catálogo de productos:

  • Alta, edición y eliminación de productos
  • Productos por unidad y por peso
  • Control de precios y stock
2

Punto de Venta

Ventas por ticket y registro automático

Sistema de ventas pensado para atención al público:

  • Venta rápida mediante ticket
  • Cálculo automático del total
  • Registro de cada operación
  • Soporte para diferentes medios de pago
3

Clientes, Fiado y Pedidos

Gestión de relaciones comerciales

Herramientas gestion de clientes:

  • Registro de clientes
  • Manejo de cuentas corrientes (pedidos)
  • Historial de anticipos

Características Destacadas

Gestión Comercial

Control total de ventas, productos y clientes.

Reportes de Ganancia

Cálculo de ganancias diarias y control de ingresos.

Clientes y Fiado

Manejo de cuentas corrientes y pedidos personalizados.
